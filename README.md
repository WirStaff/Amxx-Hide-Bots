# Hide Bots

AMX Mod X модуль для скрытия ботов из счетчика A2S/Steam API.

## Помощь

Помощь, обновления и обсуждение: [Telegram группа](https://t.me/wirstaff_inc).

## Реклама

Раскрутка серверов через мастер-сервер Steam: [steam-boost.ximply.ru](https://steam-boost.ximply.ru).

## Установка

Скопируйте бинарный файл модуля в:

```text
addons/amxmodx/modules/
```

Затем включите модуль в:

```text
addons/amxmodx/configs/modules.ini
```

Добавьте строку:

```text
hide_bots
```

Альтернативный вариант: скомпилируйте и установите AMXX плагин:

```text
amxx/scripting/HideBots.sma
```
